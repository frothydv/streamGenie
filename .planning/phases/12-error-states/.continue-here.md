## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Playwright cache-buster mismatch | content.js appends `?_cb=timestamp` to profile URLs; strict `===` comparison in route handler misses them, returns empty body, causes `res.json()` to fail with "Unexpected end of JSON input" | advisory | Always use `.split('?')[0]` or `startsWith` when comparing intercepted URLs in Playwright tests |
| waitForProfileCycle exits too early on stale path | When cache is within TTL, `loadProfile()` applies it immediately then does a background CDN refresh. `waitForProfileCycle` exits on `refs:` before the background refresh fails and `profileStaleWarning` is set. | advisory | Use `page.waitForFunction(() => el?.innerText.includes('CDN unreachable'))` instead of the generic poller for stale-cache assertions |
