    wiki.js:
    - fetchWikiItems(wikiUrl: string) → Promise<Array<{name: string, description: string}>>
    - Fetches the URL with node:fetch (built-in Node 18+); times out after 10 seconds (AbortController + setTimeout)
    - Strips HTML tags from the response text using a simple regex (/<[^>]+>/g → '')
    - Extracts lines that look like item entries: non-empty lines with 3–100 chars that don't look like navigation/headers
    - Returns up to 200 items; each has name (first word group before ':' or '—' or the whole line) and description (remainder or empty string)
    - If fetch fails or times out, returns [] (never throws — wiki is opportunistic per D-07)
    - Exports { fetchWikiItems }

    crop.js:
    - rescaleBbox(bbox, originalW, originalH): rescales bbox from Claude's 1568px-wide space to original dimensions. resizedW = 1568, resizedH = Math.round(1568 * originalH / originalW). Apply scaleX = originalW / 1568, scaleY = originalH / resizedH. Returns new bbox object with integer x, y, w, h.
    - cropFrame(framePngBuf, bbox, padding=8): uses pngjs PNG.sync.read, extracts RGBA pixels for the padded bbox region (clamped to frame bounds), returns a new PNG buffer (PNG.sync.write) and { x1, y1, cropW, cropH } for the actual crop coordinates
    - Exports { rescaleBbox, cropFrame }
  </behavior>
  <action>
    Create scripts/ai-populate/lib/wiki.js. The HTML-to-text extraction does not need to be sophisticated — it is opportunistic. A simple regex strip of HTML tags plus line splitting is sufficient. The item detection heuristic: keep lines that are 3–100 chars, not all-caps (likely headers), not starting with special chars like '#[]()'. Group into { name, description } by splitting on ' — ' or ': ' (first occurrence).

    Create scripts/ai-populate/lib/crop.js. Use pngjs for all PNG operations — do NOT use canvas. The crop extraction loop: for (let cy = y1; cy < y2; cy++) for (let cx = x1; cx < x2; cx++) copy 4 bytes from src at (cy * frameW + cx) * 4 to dst. Write to a new PNG with width = cropW, height = cropH.
