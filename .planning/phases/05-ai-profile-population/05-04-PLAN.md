<objective>
Add the developer override UI to the popup and the corresponding message handler in the content script. When a user pastes a raw GitHub branch URL, clicking Apply loads that profile directly into the running content script — allowing live testing of an AI-generated branch profile without merging it.

Purpose: This is the testing loop for the AI pipeline. Without it, the user cannot test whether AI-generated triggers actually match in live stream conditions.
Output: popup.html gets a dev URL input; popup.js handles apply; content.js handles load-dev-profile message
</objective>
